# Macro Regime Model (Gold vs Nifty)

## Indicator
Gold/Nifty ratio + TSI (25,13,13) on Monthly timeframe

## Interpretation

### TSI > 0 (Risk-Off)
- Gold outperforming
- Reduce aggression
- Smaller position sizes

### TSI < 0 (Risk-On)
- Nifty outperforming
- Trend systems work
- Add aggressively

## Use Case
- NOT for entry/exit
- ONLY for position sizing + aggression

## Advanced Version
(Gold × USDINR) / Nifty
