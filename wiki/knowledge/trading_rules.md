# Trading Rules

## Elder Impulse System
- Green: EMA rising + MACD rising
- Red: EMA falling + MACD falling
- Blue: Otherwise
## Core Rules
### Going Long
- Never Long on red bar
- long : Bar Green (Impulse Green) + Close Above 20 DMA
- Stop loss: Parabolic SAR / Close Below 20 EMA. 
-Position Size to be decided on a matrix of fii flow * Div Yield Percentile * Nifty breadth Percentile.
-As I have issues with going long I can trade Futures / Options and Even Cash Niftybees to sensitise myself towards long trades.
### Going Short
- Never Short on green bar
- Short : Bar Red (Impulse Red) + Close Below 20 DMA
- Stop loss: Parabolic SAR / Close Above 20 EMA. 
-Position Size to be decided on a matrix of fii flow * Div Yield Percentile * Nifty breadth Percentile.
## Instruments for trading
Nifty Futures,Options and Cash market Niftybees etc.
